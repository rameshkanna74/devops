# Quick Example

Suppose you want to check five servers.

Without Python:

```text
Login Server 1
Check status

Login Server 2
Check status

Login Server 3
Check status

...
```

With Python:

```python
servers = [
    "web01",
    "web02",
    "web03",
    "db01",
    "db02"
]

for server in servers:
    print(f"Checking {server}")
```

Output:

```text
Checking web01
Checking web02
Checking web03
Checking db01
Checking db02
```

This small example demonstrates the power of automation.

# Core Concepts

Before writing code, understand three important ideas.

---

## Program

A program is a set of instructions.

Example:

```python
print("Starting Backup")
print("Backup Complete")
```

Output:

```text
Starting Backup
Backup Complete
```

---

## Source Code

The instructions you write are called source code.

Example:

```python
print("Hello")
```

This line is source code.

---

## Interpreter

Python is an interpreted language.

Flow:

```text
Python Code
      |
      v
Python Interpreter
      |
      v
Program Output
```

Example:

```python
print("DevOps")
```

Output:

```text
DevOps
```

The interpreter reads the code and executes it.

# Syntax

Python uses indentation to define code blocks.

Correct:

```python

    print("Hello")
```

Incorrect:

```python

```


Indentation matters.

# Debugging Lab

## Broken Code

```python
Print("Hello")
```

Error:

```text
NameError: name 'Print' is not defined
```

---

## Root Cause

Python is case-sensitive.

```python
Print
```

is different from

```python
print
```

---

## Fix

```python
print("Hello")
```

Output:

```text
Hello
```

---

## Explanation

Built-in functions must be written exactly as defined.

Python treats uppercase and lowercase letters differently.

# Common Beginner Mistakes

## Mistake 1

Wrong:

```python
Print("Hello")
```

Correct:

```python
print("Hello")
```

---

## Mistake 2

Wrong:

```python
print(Hello)
```

Correct:

```python
print("Hello")
```

Reason:

Strings require quotation marks.

---

## Mistake 3

Wrong:

```python
if True:
print("Hi")
```

Correct:

```python
if True:
    print("Hi")
```

Reason:

Indentation is mandatory.
